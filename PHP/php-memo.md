<?php echo "出力内容"; ?> ・・・・基本形。echoが出力命令。
$変数名=代入値;  ・・・・・変数にデータを代入。
$変数名=上書き値;  ・・・・・上書き。
//コメント　　・・・・コメント
echo 変数.文字列;  ・・・・連結。変数や文字列の連結の際はドットを用いる。
※$変数名.="文字列";  で連結式を省略できる。
※""（ダブルクォーテーション）で文字列を囲む場合、中の変数を{}で囲むと変数展開してくれる。
if(条件式){実行式}　　・・・if文。
else{実行式}　　・・・・すべてFALSEの時。
else if(条件式){実行式}　　・・・条件分岐
&&と||はそれぞれ（且つ）と（または）の意味がある。

\n ・・・・・改行

switch(条件式){
   case 値:
   echo 出力値
  break;
  default:
  echo 出力値

$配列名 = array(値１, 値2, ・・・);　　・・・配列の基本。

$配列名[] = 値;　・・・・・・・・・配列の末尾に値を追加。[]の中の番号を指定するとその番号の値が上書き。
$配列名 = array('キー名' => '値１', ・・・);　・・・連想配列。インデックス番号ではなくキーと呼ばれる文字列などの値を指定。矢印はイコールと不等号の組み合わせ。
echo $配列名["キー名"]; ・・・・・連想配列からデータの取り出し。
配列名[キー] = 値;   ・・・・連想配列にデータの追加。

for ($変数名=初期値; 条件式; 更新値){echo 出力値;}  ・・・for文の基本。更新値の例として、$変数目++など。

  echo $i;
 echo "<br>";　　がある。これは501以上の数字を表示したくない時。その周だけ抜けるcontinueもあるので覚えておく。

foreach(配列 as $変数){処理式;}  ・・・・・foreach文。配列を順番どおり記述。
foreach(配列 as キー変数 => 値変数){処理式;}　　・・・・連想配列の場合のforeach文。

関数名(引数);　　　　　　　　　　　　　・・・・・・関数に値を渡す場合。

・PHP　中級編
＜オブジェクト指向について＞
・オブジェクトとは「もの」、指向とは「中心に」という意味で、簡単にいうとものを中心に考えるという意味。
・クラスは設計図。
・class class名{クラスの内容}　　・・・・・class名は必ず大文字で始める。例、class Menu{}
・クラス（設計図）を元に生成された実体を「インスタンス」と呼ぶ。
new クラス名()　・・・・・これで生成。
$変数名 = new クラス名();   ・・・生成したインスタンスを変数に代入。
※インスタンスの生成はクラスの外で行う。

・クラスの内容
・プロパティ（料理名や値段などのデータ）とメソッド（処理内容、計算など）を書く。
public $プロパティ名  ・・・・プロパティの定義。

インスタンス->プロパティ名　　・・・インスタンスのプロパティにアクセス。
例、$curry -> name = "CURRY";      ・・・・・インスタンスの名前に代入。
例、echo $curry -> name;　　　　　　・・・・・・インスタンスの名前を出力。

・メソッドとはそれぞれのインスタンスが持つ関数のこと。
